Vector initialization #2ÔÉÅ

This example contains a refactored version of the vector initialization kernel which demonstrate how the kernel would have been written in practice. Full source code of the example is shown below: 

#include <cassert>
#include <iostream>

#include <sycl/sycl.hpp>

using namespace sycl;

int main(int argc, char *argv[]) {
  const uint32_t N = (argc > 1 ? atoi(argv[1]) : 1000);
  std::vector<uint32_t> v(N);

  buffer<uint32_t, 1> buf_v{v.data(), range<1>{v.size()}};

  queue q;

  q.submit([&](handler &h) {
     accessor acc{buf_v, h, write_only, no_init};
     h.parallel_for(range<1>{N}, [=](id<1> idx) { acc[idx] = 42; });
   }).wait();

  host_accessor acc{buf_v, read_only};
  for (uint32_t i = 0; i < N; i++) assert(acc[i] == 42 && "Wrong value in v!");

  return 0;
}

In this version, we don‚Äôt use a dedicated variable to store the command group lambda function as it can be directly passed into the submit() method of the queue. Also, this example calls the wait() method on the sycl::event object returned by the submit() method (which only waits till that event finishes execution where as calling wait() on the queue will make the SYCL backend to wait till all the events in the queue to finish).
