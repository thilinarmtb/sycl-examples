Vector initialization kernel #2ÔÉÅ

This example contains a refactored version of the vector initialization kernel which demonstrate how the kernel would have been written in practice. There is no need to have a separate variable to store the command group as it can be directly passed into the submit() method of the queue. Also, this example calls the wait() method on the sycl::event object returned by the submit() method (which only waits till that event finishes execution where as calling wait() on the queue will make the SYCL backend to wait till all the events in the queue to finish).
